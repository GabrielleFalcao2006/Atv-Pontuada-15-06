# Hospital Vida & Saúde - Portal Institucional

Este projeto foi desenvolvido como parte de uma atividade prática de ReactJS com foco em roteamento.

## Sobre o Projeto
O site institucional do **Hospital Vida & Saúde** foi construído utilizando React e `react-router-dom` para gerenciar a navegação entre as diferentes seções da instituição.

## Páginas Implementadas
1. **Home**: Página inicial com destaques do hospital.
2. **Quem Somos**: Informações sobre a história, missão e valores.
3. **Serviços**: Lista de serviços médicos, exames e atendimentos.
4. **Corpo Clínico**: Apresentação dos profissionais e suas especialidades.
5. **Contato**: Informações de contato e formulário de mensagem.
6. **Blog/Novidades**: Dicas de saúde e notícias institucionais.

## Como Executar o Projeto

### Pré-requisitos
- Node.js instalado
- npm ou yarn

### Instalação
No terminal, na raiz do projeto, execute:
```bash
npm install
```

### Execução
Para iniciar o servidor de desenvolvimento:
```bash
npm run dev
```
O projeto estará disponível no endereço indicado no terminal (geralmente `http://localhost:5173`).

## Tecnologias Utilizadas
- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [React Router Dom](https://reactrouter.com/)
- Vanilla CSS para estilização
